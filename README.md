# Generative Semantic Mesh Relaying for Disaster-Resilient Communication Beyond Cellular Networks

This repository contains notebooks, figures, and outputs supporting the paper **"Generative Semantic Mesh Relaying for Disaster-Resilient Communication Beyond Cellular Networks."**

The project studies a Generative Semantic Mesh (GSM) architecture for disaster-resilient communication beyond cellular networks. GSM replaces raw emergency multimedia transmission with compact semantic descriptors, encrypted device-to-device relay, and generative reconstruction at the destination.

## Repository Contents

- `01_d2d_mesh_simulation.ipynb`  
  Notebook for D2D mesh simulation and relay-performance visualization.

- `fig_*.svg`  
  Visual outputs used for network topology, delivery/latency, and energy-overhead analysis.

## Status

This repository currently contains the D2D mesh simulation part. The semantic reconstruction evaluation notebook and its results will be added next.
